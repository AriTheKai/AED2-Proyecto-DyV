# рҹ“ҡ Enunciado del problema.

Dada una cadena $A$ con $n$ caracteres y un conjunto $S$ de 5 caracteres distintos, hay que encontrar todas las subcadenas de $A$ formadas por 3 elementos de $S$ sin repetir. HabrГЎ que devolver como soluciГіn el nГәmero de subcadenas y su posiciГіn en la cadena $C$. Por ejemplo, si

$A = abbfabcddfcbbade$ <br> $n=16$

Si consideramos un conjunto de cinco caracteres <br>$S={a, b, c, d, e}$

La soluciГіn es $4$, en las posiciones $5, 6, 13$ y $14$ <br> <br> <br>

# рҹ–ҚпёҸ DiseГұo.

Al diseГұar el algoritmo que resuelve este problema nos hemos basado en dos pseudocГіdigos previamente creados por nosotros, uno **iterativo** y otro con la estrategia de вҖң***Divide y VencerГЎs***вҖқ <br> <br> <br>

## рҹҗҢ DiseГұo Iterativo.

Para una cadena $A$ de $n$ caracteres, el algoritmo va agrupando los caracteres de 3 en 3 y comparando que:

- Ninguno estГ© repetido.
- Todos los caracteres pertenezcan a $S$.

Una vez hechas estas comprobaciones вҖ”si Г©stas se cumplenвҖ” se aГұadirГЎ el Гӯndice a la lista de subcadenas y se incrementa en $1$ el contador.

```cpp
/*
* Entrada:
*   - A: cadena de caracteres
*   - n: entero, longitud de A
*   - S: conjunto de 5 caracteres
* Salida:
*   - subcadenas: nГәmero de subcadenas que cumplen las condiciones
*   - posiciones: lista de las posiciones iniciales de subcadenas
*/

FUNCIГ“N encontrarSubcadenasIterativo(A, n, S):
        Lista<Entero> posiciones = VacГӯo
        Entero subcadenas = 0

        // Si n=16, vamos del carГЎcter 0 al 13, pues solo pueden formarse grupos de 3 en esas
        // posiciones 
        PARA c DESDE 0 HASTA n-3 HACER:
                Caracter a1 = A[c]
                Caracter a2 = A[c+1]
                Caracter a3 = A[c+2]

                // CondiciГіn 1: Que no haya duplicados
                SI (a1 == a2) O (a1 == a3) O (a2 == a3) HACER:
                        SEGUIR
                FIN SI

                // CondiciГіn 2: Que todos pertenezcan a S
                SI (a1 NO PERTENECE A S) O (a2 NO PERTENECE A S) O (a3 NO PERTENECE A S) HACER:
                        SEGUIR
                FIN SI
                // Si ambas condiciones se cumplen, contamos la subcadena y almacenamos su Гӯndice

                posiciones.aГұadir(c)
                subcadenas++

        FIN PARA

        DEVOLVER subcadenas, posiciones
FIN FUNCIГ“N
```

## рҹҰ” DiseГұo DyV .

El algoritmo de **Divide y VencerГЎs** se basa en la idea de:

> *Coger una cadena, dividirla a la mitad y comprobar las condiciones en ambas subcadenas; seguidamente, realizar las mismas comprobaciones en la subcadena del centro.*
> 

Las condiciones a comprobar son:

- NingГәn elemento estГ© repetido.
- Todos los caracteres pertenezcan a $S$.

```cpp
FUNCIГ“N encontrarSubcadenasDyV(A, inicio, fin, S)

        Lista<Entero> posiciones = VacГӯo
        Entero subcadenas = 0

        // CASO BASE: Si la longitud de la cadena es menor que 3, no se pueden formar subcadenas de 3 caracteres
        Entero n = fin - inicio + 1
        SI n < 3 HACER:
                DEVOLVER subcadenas, posiciones
        FIN SI

        // DIVIDIR Y VENCER: Dividir la cadena en dos mitades y resolver recursivamente cada mitad
        Entero medio = inicio + n / 2
        Entero subcadenasIzquierda, posicionesIzquierda = encontrarSubcadenasDyV(A, inicio, medio-1, S)
        Entero subcadenasDerecha, posicionesDerecha = encontrarSubcadenasDyV(A, medio, fin, S)

        // Comprobar subcadenas que cruzan el punto medio
        Entero subcadenasMedio = 0
        Lista<Entero> posicionesMedio = VacГӯo

        PARA c DESDE medio-2 HASTA medio-1 HACER:

                // Asegurarse de que no se salga de los lГӯmites de la cadena
                SI c < inicio O c + 2 > fin HACER:
                        SEGUIR
                FIN SI

                Caracter a1 = A[c]
                Caracter a2 = A[c+1]
                Caracter a3 = A[c+2]

                SI (a1 == a2) O (a1 == a3) O (a2 == a3) HACER:
                        SEGUIR
                FIN SI

                SI (a1 NO PERTENECE A S) O (a2 NO PERTENECE A S) O (a3 NO PERTENECE A S) HACER:
                        SEGUIR
                FIN SI

                posicionesMedio.aГұadir(c)
                subcadenasMedio++

        FIN PARA

        // Combinar resultados y devolverlos
        subcadenas = subcadenasIzquierda + subcadenasDerecha + subcadenasMedio
        posiciones = posicionesIzquierda + posicionesDerecha + posicionesMedio
        DEVOLVER subcadenas, posiciones

FIN FUNCIГ“N
```

# рҹ”Қ AnГЎlisis teГіrico.
El tiempo del algoritmo en el que se basa este codigo viene dado por la ecuaciГіn: [Teniendo en cuenta que _inicio_ = 'i'  y  _fin_ = f]
<div align="center">
$n = f - i + 1$
</div>
Al dividirlo en $\lfloor \frac{(i+f)}{2} \lfloor$ se crean dos subproblemas, el izquierdo y el derecho. De esta manera tenemos que:
$t(n) = t(n_1) + t(n_2)+ f(n) \rarrow t(n)$

```cpp
string generarMejorCaso(int longitud) {

    string cadena;

    for (int i = 0; i < longitud; i++)
    {
        char c = 'a'; // Genera solo el caracter 'a'
        cadena += c;
    }

    return cadena;

}
```

```cpp
string generarPeorCaso(int longitud) {

    string cadena;

    for (int i = 0; i < longitud; i++)
    {
        char c = 'a' + (i % 5); // Genera caracteres ciclicos de 'a' a 'e'

    }
```
